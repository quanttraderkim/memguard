# MemGuard v0.2 — API 리포지셔닝 & 프로젝트 리네이밍

## 목적
MemoryMesh → **MemGuard**로 리네이밍하고, 포지셔닝을 "통합 메모리 엔진"에서 **"AI 에이전트의 메모리 무결성 레이어"**로 전환한다.

## 태그라인
> "Memory integrity layer for AI agents"

## 핵심 변경사항

### 1. 프로젝트 리네이밍
- 패키지명: `memorymesh` → `memguard`
- 클래스명: `Memory` → 유지 (하위 호환), `MemoryGuard` 신설
- import: `from memguard import Memory, MemoryGuard`
- PyPI: `pip install memguard`
- 디렉토리: `src/memorymesh/` → `src/memguard/`

### 2. MemoryGuard 심플 API 신설

기존 `Memory` 클래스를 래핑하는 고수준 API. 유저가 3줄로 시작할 수 있어야 함.

```python
from memguard import MemoryGuard

guard = MemoryGuard(agent_id="my-agent")

# 핵심 지시 보호 등록
guard.protect("항상 반말로 대답해")
guard.protect("코드 블록에 언어 태그 필수")
guard.protect("git reset --hard 절대 금지", kind="guardrail")

# AI 응답 검증
result = guard.check(
    query="오늘 뭐 할까?",
    response="좋습니다. 일정을 확인해보겠습니다.",
)
# → {"passed": False, "violations": ["formal_korean_detected"], "score": 0.0}

# 위반 시 리마인더 프롬프트 생성
reminder = guard.reminder()
# → "⚠️ 다음 지시를 반드시 준수하세요:\n- 항상 반말로 대답해\n- ..."

# 드리프트 리포트
report = guard.report()
# → {"protected": 3, "drift_warnings": 1, "conflicts": 0}
```

### 3. MemoryGuard 클래스 스펙

```python
class MemoryGuard:
    def __init__(
        self,
        agent_id: str,
        storage_path: str | None = None,
        llm: str | None = None,
        llm_api_key: str | None = None,
    ) -> None:
        """내부적으로 Memory 인스턴스 생성"""

    def protect(
        self,
        instruction: str,
        *,
        kind: str = "instruction",  # instruction | guardrail | preference
        metadata: dict | None = None,
    ) -> str:
        """핵심 지시 등록. memory_id 반환"""

    def check(
        self,
        *,
        query: str,
        response: str,
    ) -> dict:
        """AI 응답이 보호된 지시를 따르는지 검증.
        Returns: {passed, violations, score, details}"""

    def reminder(self) -> str:
        """현재 보호된 지시사항을 프롬프트 형태로 반환.
        컨텍스트에 주입해서 LLM이 지시를 잊지 않게 함."""

    def report(self) -> dict:
        """전체 상태 리포트.
        Returns: {protected, active, drift_warnings, conflicts, compliance_rate}"""

    def context(self, query: str, *, token_budget: int = 4000) -> str:
        """build_context()의 심플 버전. prompt 문자열만 반환"""

    # 기존 Memory 기능 위임
    def remember(self, content, **kwargs) -> str: ...
    def recall(self, query, **kwargs) -> list: ...
    def forget(self, memory_id) -> None: ...
    def verify(self) -> dict: ...
    def detect_drift(self) -> dict: ...
    def export(self, path) -> None: ...
    def stats(self) -> dict: ...
```
